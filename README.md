# 🔧 Tornería Yireh

Sitio web institucional para un taller de tornería y mecanizado industrial en José C. Paz, Buenos Aires.

## 📋 Descripción

Página web one-page para Tornería Yireh. Incluye presentación de servicios, sección "Sobre Nosotros", FAQ, testimonios, ventajas, proceso de trabajo y contacto directo vía WhatsApp.

## 🛠️ Tecnologías

- HTML5
- CSS3 (con variables y animaciones)
- JavaScript (interacciones y scroll suave)
- Font Awesome (íconos)
- Google Fonts (Roboto)
- Schema.org (datos estructurados para SEO)
